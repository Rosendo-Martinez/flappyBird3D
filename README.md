# Flappy Bird 3D
A game I made to practice and develop my game development skills. <br>
I did not use a game engine. I made it with OpenGL. <br>
I made this game 100% by myself. <br>
<br>
Tech Used:
- C++
- OpenGL
- GLM
- GLFW
